Ingresar Código ID

 

Ingresar Código ID es una tarea común en Among Us, disponible en MIRA HQ y ubicada en Admin.


Jugabilidad

En Ingresar Código ID, el jugador debe deslizar hacia arriba la tarjeta de la cartera para ver su código de identificación. Deben ingresar el código en el panel correctamente para completar la tarea. Si lo ingresan incorrectamente, deberán volver a intentarlo.


Curiosidades

  • Ingresar Código ID es la contraparte de MIRA HQ de la tarea Utilizar Tarjeta, completada en The Skeld y Polus.

  • El código de identificación que ingresa el jugador es generado aleatoriamente por el juego y es diferente cada vez.
